# 🧠 آموزش‌یار پرو

یک سایت آموزشی کامل با بیش از ۲۰ دوره برای کودکان و نوجوانان.

## ✨ امکانات

- **ثبت‌نام هوشمند** با تشخیص سطح بر اساس سن
- **۲۰+ دوره آموزشی** در ۴ سطح: مبتدی، متوسط، پیشرفته، حرفه‌ای
- **داشبورد شخصی** با نمایش پیشرفت
- **بازی‌های ریاضی** تعاملی (جمع و ضرب)
- **جدول ضرب کامل** با بازی تمرینی
- **پنل مدیریت** با رمز عبور و امکان حذف کاربران
- **دفترچه عمومی** برای اشتراک‌گذاری یادداشت‌ها
- **طراحی واکنش‌گرا** برای تمام دستگاه‌ها

## 🚀 شروع

1. فایل‌ها را روی گیت‌هاب آپلود کنید.
2. سایت را باز کنید.
3. ثبت‌نام کنید و از دوره‌ها لذت ببرید!
